# Perk/Branch-Icons Planung

## Projekt-Kontext
**Tower Defense Game - Bio-Organic Theme + Branch-System**
- **6 Tower-Typen** mit je **2 Perk-Spezialisierungen** = **12 Icons total**
- **UI-Integration** für Upgrade-Panel
- **Farbpalette**: Muted Colors mit dezenten Glows

## Farb-Mapping pro Tower
- **Indica** (Grün/Teal): `#5a8a5a` - Rupture, Penetrator
- **Sativa** (Gelb): `#d4a74a` - Shrapnel, Focus  
- **Support** (Blau/Cyan): `#6be8ff` - Cryo, Toxin
- **Sniper** (Sand-beige): `#d4a74a` - Pierce, Weakpoint
- **Flamethrower** (Orange): `#b87450` - Napalm, Pressure
- **Chain** (Blau-Violett): `#8b7aa6` - Storm, Arc

## Icon-Spezifikationen
- **Größe**: 64px PNG
- **Stil**: Clean, Bio-organic
- **UI-Purpose**: Upgrade-Panel Integration
- **Lesbarkeit**: Optimiert für 32-64px Display

## Perk-Icon Details

### Indica-Perks (Heavy Cannon)
1. **Rupture**: Exploding shell/splinter pattern
2. **Penetrator**: Piercing bolt through armor crack

### Sativa-Perks (Speed/Slim)
3. **Shrapnel**: Fragment cloud/metal pieces
4. **Focus**: Scope/crosshair precision

### Support-Perks (Energy/Control)
5. **Cryo**: Snowflake/frost ring (cool blue)
6. **Toxin**: Drop/skull poison effect

### Sniper-Perks (Long Range)
7. **Pierce**: Arrow through target
8. **Weakpoint**: Skull crosshair (precision hit)

### Flamethrower-Perks (Fire/Area)
9. **Napalm**: Flask/puddle fire
10. **Pressure**: Nozzle/PSI gauge

### Chain-Perks (Lightning)
11. **Storm**: Cloud with lightning bolt
12. **Arc**: Forked lightning pattern
