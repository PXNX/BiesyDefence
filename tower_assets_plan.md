# Tower Assets Creation Plan

## Overview
Creating missing tower assets for Sativa and Support towers (Level 2 & 3) following bio/organic theme with 64-128px PNG format.

## Visual Design Guidelines
- **Background**: Dark bio theme (#010b04, #041b04)
- **Style**: Organic/futuristic with subtle glows
- **Resolution**: 64-128px PNG sprites
- **Grid**: 48px tiles, so towers should fit within ~80px bounds
- **Theme Colors**: 
  - Bio greens: #5a8a5a, #a8f57f, #80f8a3
  - Cyan accents: #6be8ff, #5a9a9a
  - Warning orange: #d4a74a, #b87450

## Tower Specifications

### Sativa Tower (Level 2 & 3)
**Theme**: Schlanke Silhouette, gelbe Akzente, "schnell" lesbar (Fast, readable, yellow accents)

**Level 2**: 
- Enhanced slim profile
- Yellow accent details on barrel/body
- Slightly longer/tapered barrel for "speed"
- Clean, fast-appearing silhouette

**Level 3**: 
- Doppel-/Triple-Barrel configuration
- Energy glow effects around barrels
- Yellow highlights intensified
- Subtle motion/blur effect suggestion

### Support Tower (Level 2 & 3)
**Theme**: Energie-/Kontrolloptik, Blau/Cyan (Energy/control optics, Blue/Cyan)

**Level 2**:
- More technological appearance
- Enhanced cyan energy cores/details
- Additional antenna/dish elements
- Control panel aesthetics

**Level 3**:
- Sichtbarem Pulsring/Anchor-Net-Akzent (Visible pulse ring/anchor net accent)
- Large pulsing energy ring around base
- Network/connection visual elements
- Enhanced cyan glow effects

## File Naming Convention
- `tower_sativa_level2.png`
- `tower_sativa_level3.png` 
- `tower_support_level2.png`
- `tower_support_level3.png`

## Creation Process
1. Design each level with progression visual cues
2. Maintain consistency with Level 1 existing assets
3. Export as 64-128px PNG with transparency
4. Test integration with existing game rendering
